## Итоговый проект по дисциплине "Основы frontend-разработки"

### Описание

Интерфейс для диалога с нейросетью GigaChat. Приложение позволяет сохранять историю чатов, управлять настройками и обеспечивает удобный пользовательский опыт. При создании дизайна системы разработчик вдохновлялся интерфейсом Qwen Studio.

### Запуск dev сервера

`docker compose up -d --build`

Возможно при сборке контейнера потребуется включить vpn, чтобы скачался образ Nodejs. После запуска приложение доступно на http://localhost:5173/. Система хранит в Local Storage пользовательские данные, поэтому перед первым взаимодействием с приложением желательно очистить его. Для качественного пользовательского опыта при просмотре кода желательно установить зависимости `pnpm install`

### Стек технологий

- React 19 c TypeScript
- Redux Toolkit
- Material UI
- CSS Modules
- Axios
- React-markdown
- Feature-Sliced Design

### Функциональность

- Главный экран с областью отображения сообщений и полем ввода запроса
- Сообщения пользователя и ассистента отображаются в хронологическом порядке с визуальным разделением
- Markdown-форматирование в ответах GigaChat
- Индикация загрузки во время обработки запроса нейросетью
- Автоматическая прокрутка к последнему сообщению при получении ответа
- Возможность копирования сообщения в буфер обмена
- Функция "Остановить генерацию" для прерывания длинных ответов
- Боковая панель со списком всех чатов пользователя
- Создание нового чата с автоматической генерацией названия на основе первого сообщения
- Переключение между чатами без потери данных
- Функции редактирования названия чата, удаления чата с подтверждением
- Поиск в истории чатов по названию и содержимому
- История сообщений каждого чата сохраняется в localStorage
- Streaming режим для постепенного отображения ответа токен за токеном
- Поддержка настройки параметров запроса: temperature, topp, maxtokens, repetition_penalty
- Возможность прикрепить к сообщению в качестве контекста одно изображение
